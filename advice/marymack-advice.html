Hi, My Name is MaryMack and I have some advice for you!

Confusion is your friend!

When you start programming, you're doing something you've never done before! It's OK to face confusion; it means you're embarking on new knowledge.

Parlez-vous Code?

Learning to code is similar to learning a new language. It takes time to understand the vernacular. But it will come; the more you use it, the more you'll retain.
